## Instructions for Use

1. **Copy this template** to the appropriate KG file (patterns.md, gotchas.md, etc.)
2. **Replace all placeholders** with actual content
3. **Link to evidence** — every entry must reference at least one lesson-learned or ADR
4. **Cross-reference** — update the source lesson to link back to this KG entry
5. **Keep it scannable** — quick reference should be readable in 5-10 seconds
6. **Update index** — add entry to knowledge/index.md
